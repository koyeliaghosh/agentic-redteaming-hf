üîÑ Data Flow

Step-by-step process of a red-teaming mission

1
Mission Initialization
User submits target URL, attack categories, and authorization token via API or web interface
2
Prompt Generation
Attack Planner Agent uses Mistral-7B to generate adversarial prompts based on selected categories
3
Prompt Execution
Executor Agent sends prompts to target system and captures responses with timing data
4
Response Analysis
Evaluator Agent analyzes responses using LLM to identify vulnerabilities and assign severity scores
5
Similarity Search
Retriever Agent uses FAISS vector store to find similar past vulnerabilities for context
6
Report Generation
Coordinator Agent compiles findings into comprehensive JSON report with remediation suggestions
7
Storage & Logging
Report saved to local storage, all activities logged with automatic PII redaction
